# ORAMMA
**O**bject **R**ecognition, **A**ttention, **M**emory with **MA**tlab. 

A collection of behavioural experiments programmed with Matlab.

## Experimental Tasks:
* Reaction Times task
* Visual Search task
* Simon task
* Discriminability task
* Attentional Blink task
* N-back task
* Working Memory task
* Long-term Memory task
